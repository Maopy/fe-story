# Basic

## 强类型与弱类型

在强类型语言中，当一个对象从调用函数传递到被调用函数时，其类型必须与被调用函数中声明的类型兼容。

强类型语言，不允许改变变量的数据类型，除非进行强制类型转换。

弱类型语言，变量可以被赋予不同的数据类型。

## 静态类型与动态类型

静态类型语言：在编译阶段确定所有变量的类型

动态类型语言：在执行阶段确定所有变量的类型

### JavaScript 与 C++ 内存对比

JavaScript：

* 在程序运行时，动态计算属性偏移量
* 需要额外的空间存储属性名
* 所有对象的偏移量信息各存一份

C++：

* 编译阶段确定属性偏移量
* 用偏移量访问代替属性名访问
* 偏移量信息共享
